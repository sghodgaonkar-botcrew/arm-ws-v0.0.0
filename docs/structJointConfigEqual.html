Detailed Description

Equality function for JointConfig to use with unordered_set.

This struct provides an equality comparison function for JointConfig vectors, enabling proper duplicate detection in unordered containers. Uses tolerance-based comparison to handle floating-point precision issues.

Comparison Strategy:

  • Tolerance-based comparison (1e-3 radians)
  • Handles floating-point precision errors
  • Ensures robust duplicate detection

Usage:

  • Used in unordered_set for duplicate detection
  • Prevents infinite loops in workspace generation
  • Ensures unique configuration sampling
