\doxysubsection{Detailed Description}
Inverse Kinematics Solver with Reachable Workspace Support. 

This header file defines the \mbox{\hyperlink{classIKSolverWithRWS}{IKSolver\+With\+RWS}} class, which provides efficient inverse kinematics solving capabilities using pre-\/computed reachable workspace data. The solver uses a two-\/stage approach\+:
\begin{DoxyEnumerate}
\item Fast nearest neighbor search to find good initial joint configurations
\item Iterative optimization using quadratic programming to refine the solution
\end{DoxyEnumerate}

The workspace data is stored in binary files containing\+:
\begin{DoxyItemize}
\item K-\/d tree data\+: sampled workspace points with positions, orientations, and joint configs
\item Voxel grid data\+: spatial indexing structure for fast neighbor search
\end{DoxyItemize}
