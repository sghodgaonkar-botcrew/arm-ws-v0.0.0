\doxysubsection{Detailed Description}
Structure to hold nearest neighbor search results. 

This structure represents the result of a nearest neighbor search, containing information about a nearby workspace point and its distance to the query point.

{\bfseries{Usage\+:}}
\begin{DoxyItemize}
\item Return type for nearest neighbor search functions
\item Used for IK warm-\/start selection
\item Enables sorting and filtering of search results
\end{DoxyItemize}

{\bfseries{Distance Metric\+:}}
\begin{DoxyItemize}
\item Distance is typically Euclidean distance in 3D space
\item Can be extended to include rotation distance
\item Used for selecting best initial configuration for IK 
\end{DoxyItemize}
