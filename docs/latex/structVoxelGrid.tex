\doxysubsection{Detailed Description}
Structure to hold sparse voxel grid data for spatial indexing. 

This structure implements a sparse voxel grid for efficient spatial indexing of workspace points. It provides O(1) spatial lookup for nearest neighbor search.

{\bfseries{Grid Structure\+:}}
\begin{DoxyItemize}
\item Regular 3D grid with specified resolution
\item Only stores occupied cells (sparse representation)
\item Maps cell indices to nearest workspace point indices
\end{DoxyItemize}

{\bfseries{Spatial Indexing\+:}}
\begin{DoxyItemize}
\item Converts 3D positions to grid cell indices
\item Enables fast neighbor search using \char`\"{}onion ring\char`\"{} expansion
\item Reduces search complexity from O(n) to O(k) where k is number of neighbors
\end{DoxyItemize}

{\bfseries{Memory Efficiency\+:}}
\begin{DoxyItemize}
\item Only stores occupied cells to save memory
\item Uses std\+::map for sparse storage
\item Scales well with workspace size 
\end{DoxyItemize}
