Detailed Description

Hash function for JointConfig to use with unordered_set (O(1) lookup)

This struct provides a hash function for JointConfig vectors, enabling efficient O(1) lookup in unordered containers. The hash function combines individual joint values using a simple but effective hashing strategy.

Hash Strategy:

  • Combines hash values of individual joint angles
  • Uses bit manipulation for good distribution
  • Provides fast computation for real-time applications

Usage:

  • Used in unordered_set for duplicate detection
  • Enables efficient configuration comparison
  • Critical for performance in large workspace generation
