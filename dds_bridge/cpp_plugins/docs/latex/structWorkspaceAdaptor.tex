\doxysubsection{Detailed Description}
Nanoflann adaptor for \mbox{\hyperlink{structWorkspacePoint}{Workspace\+Point}} data structure. 

This adaptor allows the nanoflann library to work with our custom \mbox{\hyperlink{structWorkspacePoint}{Workspace\+Point}} data structure. It provides the necessary interface functions for building and querying a k-\/d tree over workspace points.

The adaptor maps workspace points to 3D coordinates (x, y, z) for spatial indexing, enabling fast nearest neighbor searches in the workspace.
