\doxysubsection{Detailed Description}
Workspace data types and structures for IK solver. 

This header file defines the fundamental data structures used by the IK solver with reachable workspace support. It includes\+:
\begin{DoxyItemize}
\item Type aliases for fixed-\/size vectors
\item \mbox{\hyperlink{structWorkspacePoint}{Workspace\+Point}} structure for storing sampled workspace data
\item \mbox{\hyperlink{structNearestNeighbor}{Nearest\+Neighbor}} structure for search results
\item \mbox{\hyperlink{structVoxelGrid}{Voxel\+Grid}} structure for spatial indexing
\end{DoxyItemize}

These types are used throughout the IK solver to represent workspace data, search results, and spatial indexing structures.
