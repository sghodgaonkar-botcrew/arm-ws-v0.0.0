\doxysubsection{Detailed Description}
Inverse Kinematics Model class for robotic arm control. 

This class provides functionality for\+:
\begin{DoxyItemize}
\item Loading and storing robot model from URDF
\item Forward kinematics using Pinocchio
\item Collision checking
\item Cost function, gradient, and Hessian computation for IK optimization 
\end{DoxyItemize}
