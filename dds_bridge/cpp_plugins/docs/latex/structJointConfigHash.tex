\doxysubsection{Detailed Description}
Hash function for Joint\+Config to use with unordered\+\_\+set (O(1) lookup) 

This struct provides a hash function for Joint\+Config vectors, enabling efficient O(1) lookup in unordered containers. The hash function combines individual joint values using a simple but effective hashing strategy.
